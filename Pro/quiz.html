Unit 1: Introduction to Data Processing

1. What is the primary difference between data and information?
A
Data is raw facts, while information is processed data that is meaningful
B
Data is always digital, while information is always analog
C
Data is used for decision making, while information is not
D
Data is stored in computers, while information is stored on paper
Data refers to raw facts about an object, which are meaningless until processed. Information is the outcome of processed data that is used for decision making.
2. Which of the following is NOT a characteristic of computers?
A
Speed
B
Accuracy
C
Emotional Intelligence
D
Storage Capability
Computers have characteristics like speed, accuracy, storage, consistency, and repetitiveness, but they do not possess emotional intelligence as they are machines that follow programmed instructions.
3. What is the primary purpose of Electronic Data Processing (EDP)?
A
To replace human workers entirely
B
To convert raw facts into useful information through automated processes
C
To create entertainment software
D
To design computer hardware
Electronic Data Processing (EDP) is any computer automated process that converts raw facts into useful information through a well-defined procedure.
4. Which type of computer represents data as physical quantities like voltage or distance?
A
Analog Computer
B
Digital Computer
C
Hybrid Computer
D
Mainframe Computer
An analog computers represent data as physical quantities such as voltage, distance, position, or other physical quantities. Digital computers represent data as discrete binary digits (0s and 1s).
5. Which of the following is a key benefit of using computers in business?
A
Eliminating the need for human employees
B
Improved decision making through timely and accurate information
C
Reducing the need for security measures
D
Making all business processes more complex
Computers in business provide benefits like improved customer relations, increased productivity, improved cash flows, better access to information, systems integration, and improved decision making through timely and accurate information.

Unit 2: Hardware Support in Data Processing

1. Which component is considered the "brain" of the computer?
A
Central Processing Unit (CPU)
B
Random Access Memory (RAM)
C
Hard Disk Drive (HDD)
D
Motherboard
The CPU is considered the brain of the computer where data is manipulated. It consists of the arithmetic and logic unit (ALU), control unit, and memory unit.
2. Which type of memory is volatile and loses its contents when power is turned off?
A
Random Access Memory (RAM)
B
Read-Only Memory (ROM)
C
Hard Disk
D
USB Flash Drive
RAM is volatile memory that loses its contents when the computer is turned off or loses power. ROM, hard disks, and USB drives are non-volatile storage that retain data even without power.
3. Which input device is commonly used with handheld devices and can be moved with fingers or palm?
A
Mouse
B
Trackball
C
Joystick
D
Scanner
A trackball is a variant of the mouse that can be moved on a stationary device and is rotated with fingers or palm. It is commonly used with handheld devices.
4. Which component of the CPU is responsible for performing arithmetic and logical operations?
A
Control Unit
B
Arithmetic and Logic Unit (ALU)
C
Registers
D
Cache Memory
The Arithmetic and Logic Unit (ALU) is the electronic circuitry within the CPU that performs arithmetic operations (addition, subtraction, multiplication, division) and logical operations (AND, OR, NOT, comparisons).
5. Which storage device uses flash memory and has no moving parts?
A
Hard Disk Drive (HDD)
B
Optical Disk (CD/DVD)
C
Solid State Drive (SSD)
D
Magnetic Tape
Solid State Drives (SSDs) use flash memory and have no moving parts, unlike HDDs which have spinning disks. This makes SSDs faster, more reliable, and less prone to physical damage.

Unit 3: Software Support in Data Processing

1. Which type of software manages the resources of the computer and controls hardware operations?
A
Operating System
B
Word Processor
C
Spreadsheet Software
D
Database Management System
An operating system is software that manages the resources of the computer and controls hardware operations. It acts as an interface between the hardware and the user or application software.
2. Which software is used for creating and editing text documents?
A
Word Processing Software
B
Spreadsheet Software
C
Database Management Software
D
Presentation Graphics Software
Word processing software is used for creating, editing, formatting, and printing text documents. Examples include Microsoft Word, WordPerfect, and PageMaker.
3. Which type of software acts as a translator between hardware devices and the software that uses them?
A
Device Driver
B
Utility Program
C
Application Software
D
Operating System
A device driver acts as a translator between the hardware and the software that uses the devices. It allows the operating system and other software to communicate with hardware devices like printers, keyboards, and mice.
4. Which software is used for organizing, storing, and retrieving large amounts of data efficiently?
A
Word Processor
B
Spreadsheet Software
C
Database Management System (DBMS)
D
Presentation Graphics Software
A Database Management System (DBMS) is software that organizes, stores, and retrieves large amounts of data efficiently. It provides facilities for data entry, retrieval, updating, and administration of databases.
5. Which of the following is an advantage of using application packages?
A
Quicker and possibly cheaper implementation
B
Eliminates the need for customization
C
Requires no technical expertise
D
Always fits perfectly with existing systems
Advantages of application packages include quicker and possibly cheaper implementation, maximized system design and programming, provided system documentation, portability between computer systems, and efficiency in terms of speed, storage, and accuracy.

Unit 4: Filing and Data Processing Techniques

1. Which file organization method stores records in a particular order, such as by customer number?
A
Serial File Organization
B
Sequential File Organization
C
Indexed Sequential Organization
D
Direct Random File Organization
Sequential file organization stores records in a particular order, such as by customer number. This method is identical to serial access except that the records are stored in a specific order.
2. Which type of file contains relatively permanent records that are updated periodically?
A
Transaction File
B
Master File
C
Program File
D
ASCII File
A master file is a collection of relatively permanent records that are updated periodically. Once a record has been added to a master file, it remains in the system indefinitely, though its values may change over time.
3. Which data processing technique collects transaction data over a period and processes it all at once?
A
Batch Processing
B
Online Processing
C
Real-time Processing
D
Distributed Processing
Batch processing is a data processing method in which transaction data is collected over a period (e.g., a month for payroll processing) and then processed all at once as a batch to update the master file.
4. Which file organization method uses both sequential and direct access methods?
A
Serial File Organization
B
Sequential File Organization
C
Indexed Sequential Organization
D
Direct Random File Organization
Indexed Sequential Organization uses both sequential and direct access methods. It allows sequential file manipulation as records are stored in ascending order of key field, and also enables direct access using an index.
5. Which data processing technique processes data immediately as it is captured?
A
Batch Processing
B
Online Processing
C
Distributed Processing
D
Transaction Processing
Online processing is a method of data processing where data about a single transaction is processed immediately as it is captured. This allows transactions to be entered directly into the system as they occur, updating the master file immediately.

Unit 5: Fundamentals of Databases for Data Processing

1. Which of the following is the smallest logical structure of storage in a database?
A
Field
B
Record
C
Table
D
Database
A field represents one related part of a table and is the smallest logical structure of storage in a database. It holds one piece of information about an item or subject.
2. Which key uniquely identifies each record in a table?
A
Primary Key
B
Foreign Key
C
Candidate Key
D
Alternate Key
A primary key is a field or set of fields that uniquely identifies each record in a table. No two records in the table can have the same value for the primary key, and it cannot accept null values.
3. Which database model represents real-world objects as entities and their associations as relationships?
A
Entity-Relationship Model
B
Relational Model
C
Hierarchical Model
D
Network Model
The Entity-Relationship (E-R) model views the real world as a set of basic objects (entities), their characteristics (attributes), and associations among these objects (relationships). It is used for conceptual database design.
4. Which component of a database system manages the data and provides an interface between the database and users?
A
Database Management System (DBMS)
B
Data Dictionary
C
Database Schema
D
Query Language
A Database Management System (DBMS) is a software system that manages the data in the database. It provides an interface between the database and the user or application programs.
5. Which of the following is NOT a characteristic of information?
A
Accuracy
B
Timeliness
C
Volume
D
Relevance
Information must possess qualities such as accuracy, timeliness, completeness, precision, and relevance. Volume is not a characteristic of information; rather, it refers to the quantity of data.

Unit 6: Data Communications and Networks

1. Which network type typically occupies a room or building and includes a group of PCs that share a circuit?
A
Local Area Network (LAN)
B
Metropolitan Area Network (MAN)
C
Wide Area Network (WAN)
D
Backbone Network
A Local Area Network (LAN) typically occupies a room or building and includes a group of PCs that share a circuit. LANs are used to connect devices within a limited geographical area.
2. Which layer of the OSI model is responsible for routing decisions?
A
Physical Layer
B
Data Link Layer
C
Network Layer
D
Transport Layer
The Network Layer (Layer 3) of the OSI model is responsible for making routing decisions. It determines the path that data will take to reach its destination.
3. Which protocol suite is used on the Internet?
A
TCP/IP
B
OSI
C
IPX/SPX
D
AppleTalk
TCP/IP (Transmission Control Protocol/Internet Protocol) is the protocol suite used on the Internet. It consists of five layers: Application, Transport, Network, Data Link, and Physical.
4. Which network component is responsible for moving information between networks?
A
Server
B
Client
C
Router
D
Switch
Routers are specialized devices responsible for moving information between networks. They operate at the Network Layer of the OSI model and make routing decisions based on IP addresses.
5. Which type of network uses Internet technology to provide customer access to secure corporate websites?
A
Intranet
B
Extranet
C
LAN
D
WAN
An extranet uses Internet technology to provide customer access to secure corporate websites. Extranet access is usually controlled using passwords or other authentication methods.

Unit 7: Security Considerations in Data Processing

1. Which of the following is NOT part of the CIA triad of data security?
A
Confidentiality
B
Integrity
C
Accessibility
D
Availability
The CIA triad of data security consists of Confidentiality, Integrity, and Availability. Accessibility is not part of the CIA triad, though it is related to the Availability principle.
2. Which type of attack involves carefully targeted digital messages to trick people into clicking on malicious links?
A
Phishing Attack
B
Ransomware Attack
C
Cyber-physical Attack
D
State-sponsored Attack
Phishing attacks involve carefully targeted digital messages transmitted to fool people into clicking on a link that can install malware or expose sensitive data.
3. Which security measure involves restricting access to users who have been granted access to information?
A
Access Control
B
Data Encryption
C
Email Security
D
Risk Assessment
Access Control security involves restricting access of users who have been granted access to information, thereby monitoring who has access to particular data.
4. Which security measure protects data by converting it into a coded format that can only be read with the correct key?
A
Firewall
B
Data Encryption
C
Antivirus Protection
D
Regular Backups
Data encryption protects data by converting it into a coded format that can only be read with the correct decryption key. This prevents unauthorized access to sensitive information.
5. Which security measure involves making copies of data to prevent loss in case of system failure?
A
Strong Firewall
B
Antivirus Protection
C
Regular Backups
D
Access Control
Regular backups involve making copies of data and storing them in a safe location to prevent loss in case of system failure, accidental deletion, or other disasters.

Unit 8: Information Systems in Data Processing

1. Which type of information system processes routine transactions necessary to conduct business?
A
Transaction Processing System (TPS)
B
Management Information System (MIS)
C
Decision Support System (DSS)
D
Executive Information System (EIS)
A Transaction Processing System (TPS) is an organized collection of people, procedures, software, databases, and devices used to record completed business transactions. It processes routine transactions necessary to conduct business.
2. Which type of information system provides information to managers to help them make informed decisions?
A
Transaction Processing System (TPS)
B
Management Information System (MIS)
C
Office Automation System (OAS)
D
Executive Information System (EIS)
A Management Information System (MIS) provides information to managers to help them make informed decisions. It supports the managers in the decision-making process and enhances their productivity and performance.
3. Which type of information system supports problem-specific decision making using mathematical or logical models?
A
Transaction Processing System (TPS)
B
Management Information System (MIS)
C
Decision Support System (DSS)
D
Office Automation System (OAS)
A Decision Support System (DSS) supports problem-specific decision making by applying mathematical or logical models and interactions to solve problems. It is user-friendly and highly interactive.
4. Which specialized information system manages and coordinates all resources, information, and functions of a business from shared data stores?
A
Enterprise Resource Planning (ERP)
B
Expert System
C
Electronic Commerce (E-commerce)
D
Executive Information System (EIS)
Enterprise Resource Planning (ERP) is a computer software system that manages and coordinates all resources, information, and functions of a business from shared data stores. It integrates various business processes.
5. Which type of information system provides summarized information in a convenient form to top-level managers?
A
Transaction Processing System (TPS)
B
Management Information System (MIS)
C
Decision Support System (DSS)
D
Executive Information System (EIS)
An Executive Information System (EIS) provides summarized information in a convenient form to top-level managers. It is designed to facilitate access to information quickly and effectively for senior managers.
